Instalación de ECVLib con Composer

Para instalar el framework utilizando Composer, ejecuta el siguiente comando:

composer create-project arkeber-23/ecvlib nombre_proyecto

Este comando descargará e instalará el framework del proyecto.

Para iniciar el servidor use la línea de comando:

php thor serve

Servidor iniciado en: http://localhost:8080, a evaluación de los puertos disponibles.
